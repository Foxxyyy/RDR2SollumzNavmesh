# SollumzRDR – YNV Support

`Commit: 19f8d5b05715a9f0754c96074f3b9fcb78f3d477 (August 2025)`

This fork of **SollumzRDR** adds support for **YNV navmesh files** used in Red Dead Redemption 2.

## Tutorial
A tutorial explaining how to use the plugin is available here:  
https://github.com/charlietonygene/rdr3-navmesh-streaming-redm

## Mexico Navmeshes

Navmeshes generated by **myself and BadassBaboon** for the **Mexico region** are available for public use.

These were created primarily for the **Nuevo Paraíso: The Forgotten Frontier** mod.

Download:  
https://www.mediafire.com/file/d6wcdnyvkedz1hw/Navmesh.zip/file
